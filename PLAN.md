# cross-platform-clipboard

在当前目录下，使用 Next.js + TailwindCSS + Supabase 实现跨平台剪贴板。

## 基本功能

- 用户登录和注册，每个用户只能看到自己的剪贴板内容
- 剪贴板内容创建时，可以指定过期时间，过期后的内容不再显示
- 定时任务自动清理用户剪贴板中过期的内容

## UI 要求

- 未登录时首页显示 Hero 区：项目名称、简要介绍和快速操作入口
- Apple 风格：简洁、优雅、注重细节的设计风格
- 合适的高斯模糊、半透明、毛玻璃效果，提升界面层次感和视觉效果
- 界面适应不同屏幕尺寸，包括桌面和移动设备
- 提供浅色和深色模式，满足用户的视觉偏好
